# Environment Variables for Vercel

After deploying to Vercel, add these environment variables in Vercel Dashboard → Settings → Environment Variables:

1. MONGODB_URI = mongodb+srv://<username>:<password>@cluster.mongodb.net/socialconnect
   (Get this from MongoDB Atlas after creating free cluster)

2. NEXTAUTH_URL = https://social-connect.vercel.app
   (Replace with your actual Vercel URL after deployment)
